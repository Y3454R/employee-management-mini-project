* Employee list <br>
* Add, remove employee <br>
* backend: Node.js<br>
* frontend: React<br>
To make the project ready:

- npm
  ```sh
  npm install
  ```
  To run the server:
- npm
  ```sh
  cd server
  npm start
  ```
  To run the client:
- npm
  ```sh
  cd client
  npm start
  ```
