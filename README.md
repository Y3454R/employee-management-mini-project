Fetch a backend API and show all the employees. <br>
To make the project ready:
* npm
  ```sh
  npm install
  ```
To run the server:
* npm
  ```sh
  cd server
  npm start
  ```
To run the client:
* npm
  ```sh
  cd client
  npm start
  ```
